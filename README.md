# car simulator
자동차 시뮬레이터



엔진: https://github.com/microsoft/DirectX-Graphics-Samples/blob/MicrosoftDocs-Samples/MiniEngine/readme.md



최종목표: 

자동차의 움직임을 실제처럼 정교하게 구현

렌더링은 중요하지 않고 움직임이 중요 



세부목표: 

Dirextx mini engine으로 자동차와 도로를 띄우기

바퀴의 움직임을 물리적으로 구현 

스프링의 움직임을 물리적으로 구현 

바퀴들끼리는 어떻게 연결되어 있을까

자동차 몸체와 바퀴 사이에는 어떤 상호작용이 있을까 

명령에 따라 바퀴에 어떤 힘이 가해질지에 대해 구현 

핸들 움직임 구현 



추가목표: 

자동차의 바퀴는 온전한 강체가 아님(타이어) 이걸로 인해 영향이 있을까? 

지면의 속성이 변함에 따라 어떻게 움직임이 달라질까

1주차: 

1 Directx mini engine에 대한 이해 및 기본적인 장면 출력

2 기본적인 물체의 힘(가속도)와 속도, 이에따른 위치변화에 대해 구현

3 중력과 이에 따른 상호작용에 대해 구현

4 회전중심에서 돌림힘이 가해질 때, 바닥 접지면에서 마찰력이 가해질 때의 강체의 회전에 대해 물리적으로 구현 

5 힘이 가해질 때 스프링의 움직임에 대해 물리적으로 구현



2주차: 

1 바퀴와 강체와 스프링을 이어서 서스펜션 완성하기



3주차: 

1 서스펜션 2개를 이어서 자동차 아랫부분 완성하기

2 핸들을 돌림(A키와 D키)에 따라 바퀴에 어떤 움직임이 나타나는지 구현



4주차: 

1 자동차 몸체와 서스페인을 스프링으로 이어서 자동차 완성하기



5주차: 

1 앞키(W)>엑셀, 뒷키(S)>브레이크를 입력했을 때 바퀴에 어떤 힘이 가해지고 어떤 움직임이 나타나는지 구현



6주차: 

1 타이어 구현



7주차: 

1 지면의 속성이 달라짐에 따라 어떻게 움직임이 달라지는지 구현



8주차: 

마무리
